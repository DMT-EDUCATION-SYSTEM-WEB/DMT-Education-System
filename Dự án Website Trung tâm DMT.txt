## Dự án Website Trung tâm DMT 

## TỔNG QUAN DỰ ÁN
Dự án này nhằm xây dựng một website cho trung tâm đào tạo DMT, cung cấp các chức năng quản lý học sinh, giáo viên, admin và nhân viên. 
Mục tiêu là tạo ra một hệ thống quản lý học tập trực tuyến hiệu quả, bảo mật và thân thiện với người dùng, hỗ trợ các hoạt động học tập và quản lý nội bộ.
### 1. HỌC SINH / PHỤ HUYNH

Chức năng cốt lõi:
- Xem video ghi hình buổi học (file video được update bởi admin, bảo vệ bản quyền bằng cách không cho phép tải xuống)
- Xem thời khóa biểu cá nhân (mỗi tài khoản riêng biệt lịch học khác nhau)
- Truy cập tài liệu PDF (có watermark, cho phép tải offline có giới hạn, không cho phép chỉnh sửa và tải xuống)
- Xem điểm và nhận xét đánh giá từ Giảng viên
- Tham gia khảo sát (điều kiện: đã hoàn thành bài học)
- Xem thông tin khóa học(thời lượng khóa học, học phí, thông tin giảng viê, trợ giảng)
- Dashboard theo dõi tiến độ học tập
- Tích hợp lịch Google Calendar
- Quản lý tài khoản cá nhân (đổi mật khẩu, thông tin cá nhân)
- Quản lý lịch sử học tập (xem lại các buổi học đã tham gia, các video đã xem, các file đã tải xuống)
- Tính năng nhắc nhở deadline bài tập/đề kiểm tra,... qua email/SMS
- Tính năng gửi feedback cho Giảng viên
- Tính năng theo dõi lịch sử đăng nhập trong tài khoản
- Tính năng quản lý thông báo (đọc, đánh dấu đã đọc, xóa)
- Tính năng theo dõi học phí (đã thanh toán, còn nợ)
- Lịch sử hoạt động học tập

Bảo mật:
- Tối đa 2 người/tài khoản đăng nhập cùng lúc
- Tự động logout khi login đồng thời nhiều thiết bị(3 trở lên)
- Session timeout: 2 giờ không hoạt động
- Lịch sử đăng nhập

Thời hạn: 1 năm, tự động gia hạn khi tham gia khóa mới

---

### 2. GIÁO VIÊN / TRỢ GIẢNG

Chức năng cốt lõi:
- Tạo bài tập/đề kiểm tra với deadline do Admin cài đặt
- Chấm bài và nhận xét trực tiếp trên hệ thống
- Upload tài liệu, video bài giảng
- Tạo khảo sát lấy ý kiến học sinh
- Đăng ký nghỉ phép
- Quản lý bảng chấm công
- Hệ thống notification deadline chấm bài
- Export danh sách điểm (Excel/PDF)
- Calendar tích hợp với Google Calendar
- Backup tự động nội dung bài giảng
- có bảng chấm công cho mỗi giáo viên 

Báo cáo:
- Báo cáo tiến độ giảng dạy
- Hiệu quả bài kiểm tra

---

### 3. ADMIN

Chức năng cốt lõi:
- Quản lý toàn bộ user và phân quyền
- Kiểm tra việc chấm bài của Giảng viên/TG
- Kiểm tra và phân tích khảo sát
- Quản lý vòng đời tài khoản
- Cài đặt deadline chấm điểm
- Dashboard tổng quan hệ thống
- Analytics chi tiết (học sinh, doanh thu, hiệu quả)
- Quản lý backup/restore
- Hệ thống cảnh báo bảo mật
- Quản lý maintenance mode
- Cấu hình hệ thống notification
- Quản lý template email/SMS
- Xem thống kê học sinh theo khu vực
- Quản lý bảng chấm công cho tất cả giáo viên

Báo cáo nâng cao:
- Báo cáo doanh thu theo khóa học
- Phân tích xu hướng đăng ký
- Hiệu quả giảng dạy theo Giảng viên
- Tỷ lệ hoàn thành khóa học
- Xem thống kê học sinh theo khu vực

---

### 4. NHÂN VIÊN

Chức năng cốt lõi:
- Hỗ trợ kiểm tra tình trạng học sinh, giáo viên
- Theo dõi task nội bộ
- Xử lý ticket support

Chức năng mới:
- Dashboard task management
- Live chat với học sinh/phụ huynh
- Quản lý FAQ và knowledge base
- Báo cáo customer service
- Theo dõi satisfaction rating

---

## BẢO MẬT VÀ XÁC THỰC

Yêu cầu mới:
- Two-Factor Authentication (2FA) cho Admin và Giảng viên
- Mã hóa dữ liệu nhạy cảm (AES-256)
- Password policy mạnh (8+ ký tự, đặc biệt, số)
- Rate limiting cho API calls
- CAPTCHA cho login sau 3 lần sai
- IP whitelist cho Admin
- Regular security audit logs

---

## TÍNH NĂNG HỆ THỐNG
- RBAC tối thiểu (vai trò: Admin, Giáo viên, Học sinh/Phụ huynh, Nhân viên).
- Theo dõi thay đổi quan trọng (audit tối thiểu cho điểm/chấm bài).

Notification System:
- Email notifications
- SMS notifications (quan trọng)
- Push notifications (mobile)

Search & Filter:
- Tìm kiếm thông minh (fuzzy search)
- Filter đa tiêu chí
- Sort theo nhiều cột
- Export filtered results

API & Integration:
- RESTful API đầy đủ
- Webhook cho events quan trọng
- Third-party integrations:
  - Payment gateways (VNPay, Momo)
  - Google Calendar/Outlook
  - Zoom/Teams
  - SMS providers

## KIẾN TRÚC KỸ THUẬT 

Tech Stack:
- Frontend: React.js + TypeScript + Tailwind CSS
- Backend: Node.js/Python Django + REST API
- Database: SQLserver + Redis cache
- File Storage: Cloud storage with CDN
- Real-time: WebSocket/Socket.io
- Monitoring: Application insights

Performance:
- CDN for static assets
- Database indexing
- API caching
- Image optimization
- Lazy loading
- Code splitting
